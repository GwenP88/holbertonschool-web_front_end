# Project - Advanced HTML — Techium Project

## Description

This project is an introduction to HTML5 structure and semantic markup. The goal is to build the skeleton of a multi-section webpage for a fictional digital agency called **Techium**, using only HTML — no CSS, no styling. The focus is on writing clean, valid, and semantic HTML that follows W3C standards.

## Learning Objectives

By the end of this project, you should be able to explain:

- Which guidelines to follow for HTML
- How to create the skeleton of an HTML5 page
- How to use semantic HTML tags to structure a web page
- When to use `div` vs `span`
- The semantic value of `header`, `main`, `footer`, `article`, `nav`, `section`, and `aside`
- How to use headings and why hierarchical order matters
- How to create lists in HTML
- The differences between media formats: SVG, GIF, PNG, JPG
- How to structure data in a table
- How to integrate video and audio into a webpage
- How to embed external content
- How to correctly structure an HTML page

## Page Sections

The homepage (`index.html`) includes the following sections:

| Section | Description |
|---|---|
| **Header** | Logo and main navigation |
| **Hero** | Main call-to-action banner |
| **Services** | List of services offered |
| **Works** | Portfolio showcase |
| **About Us** | Company background and culture |
| **Latest News** | Blog article previews |
| **Testimonials** | Client quotes |
| **Contact** | Contact information and link |
| **Footer** | Logo, address, social links, and legal links |

## Requirements

- A `README.md` file at the root of the project folder is mandatory
- All code must be **W3C compliant** and validate with the [W3C Validator](https://validator.w3.org/)
- The company name used throughout the project is **Techium**
- No CSS or styling — this project focuses exclusively on HTML structure
